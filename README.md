# 📚 ReadCast

**ReadCast** is a modern personal online library built with Next.js, designed for reading **EPUB books directly in the browser**.  
It was created with a special use case in mind: accessing and reading books comfortably on devices with limited browsers such as Smart TVs.

---

## ✨ Features

- 📖 Read EPUB books directly in-browser
- 🎨 Change fonts dynamically
- 🌙 Multiple reading themes (Dark / Light / Sepia)
- 🔎 Search books instantly
- 💾 Auto-save reading progress
- ⚙️ Save reader preferences (font, size, theme)
- 🖥️ TV-friendly responsive interface
- 📚 Beautiful bookshelf style homepage
- ⌨️ Keyboard / Remote navigation support

---

## 🛠️ Tech Stack

- **Next.js** (Pages Router)
- **React**
- **Tailwind CSS v4**
- **epub.js**
- **React Icons**
